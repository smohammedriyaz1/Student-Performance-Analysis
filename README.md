# 🎓 Student Performance Analysis

## 📌 Overview

This project analyzes student performance using **Python**, **Pandas**, and **NumPy**.

The project demonstrates data cleaning, validation, and exploratory data analysis (EDA).

---

## 🚀 Features

- Load student dataset
- View dataset information
- Check data types
- Find missing values
- Find duplicate records
- Convert date columns
- Detect invalid marks
- Detect invalid attendance
- Basic data analysis

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
